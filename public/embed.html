This widget can be embedded on your website using an iframe.

Example code:

<iframe 
  src="http://your-server:5643?mode=widget&theme=dark&compact=true&header=false" 
  width="100%" 
  height="600" 
  frameborder="0"
  style="border: 1px solid #ddd; border-radius: 8px;"
></iframe>

View Widget Preview

Parameters: mode (widget/full), theme (light/dark/anime), compact (true/false), header (true/false), auth (true/false)
